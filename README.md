Fiskespill

Kjør filen CanvasApp.java

Very fun game 10/10

Spillet går ut på å få så mye poeng som mulig på 60sek. Man får poeng for å fiske opp fisk, og enkelte fisker (størrelse, type) gir mer poeng.
